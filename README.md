## Favorite Code segment
Yes. It is quicksort in Haskell.

```Haskell
quicksort [] = []
quicksort (p:xs) = (quicksort lesser) ++ [p] ++ (quicksort greater)
    where
        lesser = filter (< p) xs
        greater = filter (>= p) xs
```

This is copied from [Haskell Website](https://wiki.haskell.org/Introduction#Quicksort_in_Haskell).
